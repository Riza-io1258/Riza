Instal Aplikasi di Komputer

Langkah-langkah

1. Langkah pertama buka aplikasi browser atau microsoft store.

2. Kemudian cari aplikasi yang ingin di download pada kolom pencarian.

3. Setelah itu pada aplikasi browser kalian cari salah satu situs untuk mendownload aplikasi sedangkan pada aplikasi microsoft store kalian langsung pilih aplikasi yang ingin di download

4. Lalu setelah memilih aplikasi yang ingin di download, setelah itu kalian klik download.

5. tunggu sampai selesai.
6. Apabila kalian download aplikasi melalui browser kalian, setelah kalian mendownload aplikasi, kalian harus instal aplikasi terlebih dahulu sedangkan apabila kalian mendownload aplikasi melalui microsoft store maka secara otomatis aplikasi akan langsung terinstal.
