Ubah Password dan Username di Wifi Menggunakan HP

Langkah-langkah

1. Langkah pertama sambungkan wifi terlebih dahulu ke hp agar password maupun username bisa diubah.
2. Kemudian masuk ke pengaturan pilih menu “wifi” pada hp dan masuk ke wifi yang terhubung ke hp, lalu cari tulisan gateway dan salin IPnya.

3. Setelah itu buka browser dan paste IP yang sudah di salin di kolom pencarian dan klik enter maka secara otomatis akan masuk ke halaman buat ubah username maupun password

4. Lalu masuk ke menu quick setup

5. Kemudian ubah password dan username lalu klik save, maka secara otomatis wifi akan melakukan reboot.

6. Tunggu proses reboot sampai selesai.
7. Langkah terakhir coba hubungkan hp ke wifi dengan username baru dan menggunakan password baru.
